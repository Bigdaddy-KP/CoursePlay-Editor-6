Welcome to Coursedrawer Web Page

Course Drawer is a tool for visual editing of XML files used in Courseplay Farming Simulator 2015.
Course Drawer, developed initially by Pawpouk, has been rebuilt by us to deal with the new data produced by Courseplay 4.0 in new FS2015.
You have to install .NET 4.0 to run. Only Windows

New features in version 4.0

Transform and validate old Courseplay files to FS15
Identify speed errors and correct them.
Add Courseplay initial settings in case doesn’t exist.
Reduce the size of XML file using new structure

Support procedure when errors and issues

 No support via PM!
 Bugs are handled on our issues page.
 Have a look if the bug has already been reported in the issues section on GitHub
 State the Course Drawer version you're using. You can find it in the upper corner of the screen.
 Explain the situation: what did you did? What was the goal? When did the error occur? Is the error reproducible?
